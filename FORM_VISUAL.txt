================================================================================
                            FORM VALIDATION
================================================================================

When patient clicks "Book Appointment" button:

✅ CHECK 1: Full Name
   - Must not be empty
   - Error message: "Name field is required"

✅ CHECK 2: Email Address
   - Must not be empty
   - Must be valid email format (xxx@xxx.xxx)
   - Error message: "Valid email required"

✅ CHECK 3: Phone Number
   - Must not be empty
   - Error message: "Phone number required"

✅ CHECK 4: Service Required
   - Must have selection (not "Select a service")
   - Error message: "Please select a service"

✅ CHECK 5: Message
   - No validation (optional field)

If all checks pass → Form submitted ✅
If any check fails → Form shows error message ❌

================================================================================
                          SUCCESS FLOW
================================================================================

1. Patient fills all required fields
           ↓
2. Patient clicks "Book Appointment" button
           ↓
3. Form validates (checks all fields)
           ↓
4. Button shows "Sending..." (disabled, grayed out)
           ↓
5. Form data sent to Formspree
           ↓
6. Formspree forwards to: drraghven10@gmail.com
           ↓
7. Success message appears:
   "Thank you! Your appointment request has been sent. 
    We'll contact you soon."
   
   (Green box for 3 seconds, then hides)
           ↓
8. Form fields clear for next submission
           ↓
9. Email inbox receives:
   From: Patient's Email
   To: drraghven10@gmail.com
   Subject: New Appointment Request from [Patient Name]
   Body: All form data (name, email, phone, service, message)

================================================================================
                          ERROR FLOW
================================================================================

1. Patient submits incomplete/invalid form
           ↓
2. JavaScript validation fails
           ↓
3. Form shows error message or field highlights
           ↓
4. Patient corrects the error
           ↓
5. Patient tries again

OR

1. Network error occurs during submission
           ↓
2. Red error message appears:
   "Error sending form. Please try again or call us directly."
           ↓
3. Patient can retry or call clinic directly
   Phone: +91 8107667788
